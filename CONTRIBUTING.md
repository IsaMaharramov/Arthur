# Contributing to Arthur

I welcome contributions from the community! Whether you want to add new levels, fix bugs, or optimize the raycasting engine, here is how you can help.

## How to Contribute
1. **Fork the repository.**
2. **Create a feature branch:** `git checkout -b feature/my-new-feature`
3. **Make your changes.**
4. **Commit your changes:** `git commit -m 'Add some feature'`
5. **Push to the branch:** `git push origin feature/my-new-feature`
6. **Open a Pull Request.**

## Guidelines
* **Code Style:** Please try to keep your code consistent with the existing style.
* **Documentation:** If you add a new enemy or level, please update the README.
* **Testing:** Ensure your code compiles with CMake before submitting a Pull Request.
